# KDSI House Management System (Frontend)

This is the frontend of the **KDSI House Management System**, a web-based application designed to manage staff houses at KD Sultan Ismail (KDSI). This application is built using React.js.

## Features

- **User Authentication**: Login and Register functionalities for users with roles.
- **Dashboard**: Summarized details of houses, maintenance, and guest management.
- **House Management**: CRUD operations for houses.
- **Maintenance Requests**: Submit and track maintenance complaints.
- **Guest Management**: Check-in and check-out for guests.
- **Reporting**: Generate and view monthly reports.
