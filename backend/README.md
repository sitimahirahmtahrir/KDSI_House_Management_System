# KDSI House Management System

## Overview
The KDSI House Management System is a Laravel-based web application developed to streamline the management of staff houses, maintenance requests, guest check-ins, and reports for the KD Sultan Ismail (KDSI) naval base.

## Features
- User Authentication with Role Management (Admin, User)
- CRUD Operations for House Management
- Maintenance Request Submissions
- Guest Check-In/Check-Out Management
- Dynamic Dashboards for Admins and Users
- Reporting and PDF Generation
